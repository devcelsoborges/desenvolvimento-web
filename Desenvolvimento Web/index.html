Sobre Mim

Messi Careca Vascaíno é um desenvolvedor de front-end sénior do Vasco da Gama, com mais de 10 anos de experiência a criar interfaces intuitivas e responsivas para milhões de utilizadores. Apesar de ter perdido os cabelos a programar sob pressão durante as finais de campeonato, nunca perdeu o espírito vascaíno nem o amor pelo código limpo. Conhecido por driblar bugs com a mesma habilidade com que Messi dribla defesas, Messi Careca é o maestro do HTML, CSS e JavaScript, liderando a equipa técnica do Vasco para vitórias digitais. Quando não está a desenvolver funcionalidades campeãs, está a festejar as vitórias do Vasco ou a criar memes épicos para o Twitter..
